〜 Profile 〜

アラシ（本名：千葉嵐士）/ 2002年生まれ / 天秤座 / O型

石巻出身 / 高校を卒業後、東北工業大学に入学。

中学生の時に遊びすぎて、高校受験に落ちる。ストレスで血を吐いたり、熱が出たりしたが何とか地元の高校に二次試験で合格する。

友達がやんちゃな奴しかいなかったが、自分が落ちた高校の奴らを見返すため勉強する。

親の負担を減らすため資格を取ったり、評定を５にしたりで国立大学の受験資格得て受験する。

コロナの影響で前年よりも倍率が2倍に、、その結果不合格。すべり止めで受けてた東北工業大学に入学。

好きなもの：バレー / バスケ / マンガ / アニメ / ラーメン / スマブラ
